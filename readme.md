# Logistics Tracking
[ktor](https://ktor.io/docs/welcome.html)을 이용해 택배 배송 상태를 조회하는 서비스입니다.
# Tech Specifications
- Kotlin 1.8.21
- Ktor 2.3.0

# ktlint
아래 명령어를 실행하여 ktlintFormat을 실행합니다.
```shell
./gradlew ktlintFormat
```
